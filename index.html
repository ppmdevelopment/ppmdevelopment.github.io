Learn Go — Peng AI

A fully functional AI chatbot I built from scratch using Python and Machine Learning. Peng AI teaches the ancient strategy game of Go (Weiqi) — from basic rules like Atari and Liberties to advanced tactics like Joseki, Tesuji, and Ko fights.

Built with a real ML pipeline (TF-IDF + Naive Bayes) that understands natural language. Features a live self-learning system — when the AI doesn't know an answer, it saves the question so I can train it in real time without restarting the server.
